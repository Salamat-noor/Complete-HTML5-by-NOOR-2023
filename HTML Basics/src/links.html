In HTML (Hypertext Markup Language), links are an essential element used to navigate between different web pages, resources, or
sections within a document. They allow users to interact with the content and explore related information conveniently. Links are represented using the (anchor) element and are typically displayed as clickable text or images




Syntax: The basic syntax for creating a link is as follows

< a href=" URL "> Link text < /a> 

The href attribute specifies the destination of the link. 
It can be an absolute URL (e.g., http://example.com) or a relative URL (e.g., about.html, ../index.html).

The link text is the visible part that users can click on.
Absolute URLs: Absolute URLs point to resources located on different websites or servers. They include the protocol (e.g., http://, https://) 
 followed by the domain name and the path to the resource. For example:

< a href=" https://example.com "> Visit Example < /a> 

Linking to Email Addresses: You can also create links that open the user's default email client with a pre-filled email. To achieve this, use the mailto: 
 scheme followed by the email address. For example:

< a href=" mailto:info@example.com "> Contact us < /a> 

Target Attribute: The target attribute can be used to specify how the linked content should open. Common values include:

_blank: Opens the link in a new browser tab or window.
_self: Opens the link in the same frame or window.
_parent: Opens the link in the parent frame or window.
_top: Opens the link in the full body of the window.

< a href=" https://example.com " target=" _blank "> Open in new tab < /a>
