blockquote

this tag is used when a source or a partition of text is taken from another web site to our website. and the taken text will be in the body of blockquote

< q> Short Quotations

This tag is used also instead of blockquote but this will insert quotation marks around its inner text and it will used for a short inline text. For Example, CSS

Abbreviations

This tag is useful when like HTML , CSS , Mr. words are given on the page. These type of words gives us an extra meaning when mouse is hover over them. For Example, word HTML is abbrivated and gives us HTML

Address

This tag is used when we have to put the contact information for the author of a document or an article. This will be shown italic type text.

cite

This tag defines the title of a creative work. A human name does not exist in this tag only a work name will be shown in this tag. For Wxample, A book, or a poem, or a song.

bdo , Bi-Directional Override

bdo stands for Bi-directional Override This tag is used to Override the current direction of text to right to left
