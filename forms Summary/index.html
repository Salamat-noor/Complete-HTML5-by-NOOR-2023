Input pattern attribute

The input pattern attribute specifies a regular expression that the input field's value is checked against, when the form is submitted. The pattern attribute works with the following input types:
text,
date,
search,
url,
tel,
email,
password.

min & max Attribute

The min max attribute is use to set the limitations in the input field The min and max attributes work with the following input types:
number,
range,
date,
datetime-local,
month,
time
week

The maxlength attribute is use to specify the maximum number of characters allowed in the input field.

step attribute

The step attribute is used to make a validate of given value, it is simply used to make multiples of the given value in numbers it makes the multiple of the given value. This attribute can be used together with the max and min attributes to create a range of legal values.

placeholder attribute

The placeholder attribute is used when we have to give a hint behind the input field. The input placeholder attribute specifies a short hint that describes the expected value of an input field (a sample value or a short description of the expected format). The short hint is displayed in the input field before the user enters a value. The placeholder attribute works with the following input types:
text,
search,
url,
tel,
email,
password.
